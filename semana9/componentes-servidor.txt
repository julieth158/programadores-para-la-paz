Sistema operativo:
Es el software que administra los recursos del computador. Permite ejecutar programas y controlar el hardware del sistema.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son registros que almacenan eventos y actividades del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
